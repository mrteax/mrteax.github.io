📲 一键把上传配置导入 Health Auto Export

在 iPhone 上点上面的按钮（需已装 Health Auto Export）：它会自动打开 App 并填好上传地址和鉴权，你只要勾选要同步的指标、把频率设为每天即可。

⚙️ 怎么开启「每天自动上传」（一次性设置）

1）在 Supabase 建一张表（SQL Editor 里运行，仅一次）：

create table health_imports (
  id bigint generated always as identity primary key,
  data jsonb,
  created_at timestamptz default now()
);
alter table health_imports enable row level security;
create policy "read"   on health_imports for select using (true);
create policy "insert" on health_imports for insert with check (true);

2）在 Health Auto Export 里建一个自动化（需 App 内的 Automation 功能）：

  • 新建 Automation，指标勾选 Sleep / Resting Heart Rate / HRV（可加 Steps、Active Energy 等）
  • 频率选 Daily（每天），格式 JSON，类型选 REST API
  • URL 填：
    …
  • Method 选 POST，添加 Header：
    apikey = 你的 anon key
    Authorization = Bearer 你的 anon key
    Content-Type = application/json
  • 保存。之后每天自动上传，打开本页就能看到 ✅

⚠️ 报 timeout / 超时？说明单次导出的数据太大。把它调小就好：

  • Aggregate Data 打开（按时段聚合，而不是导出每个采样点）
  • Period 选 Today 或最近 1 天，不要选 All / 全部历史
  • Aggregation Interval 选 Hours / Days（颗粒度越粗、数据越小）
  • Batch Requests 保持 关闭
  • 若仍超时，把 Request Timeout 调大到 60 秒

（上面的一键按钮已默认设成 Today + 按小时聚合，正常不会超时。）

没有 Automation 功能？也可以用 iOS「快捷指令」做一个每日自动化，读取健康数据后 POST 到同一个 URL，效果一样（免费）。

📱 怎么拿到数据

方案一：最简单 — 直接看 iPhone

打开 iPhone 健康 App，首页「摘要」就能看到：
· 睡眠 → 时长和质量
· 心脏 → 静息心率、HRV
把数字手动填到上面的表单就行。

方案二：一键导出 — Health Auto Export

App Store 搜索 Health Auto Export（免费版就够用）：

  1. 下载安装，授权 Apple Health 读取权限
  2. 选择要导出的指标：Sleep Analysis、Resting Heart Rate、Heart Rate Variability（可选加 Steps、Active Energy、VO₂ Max 等）
  3. 导出格式选 JSON，点 Export 保存 .json 文件
  4. 把文件传到这台设备，拖到本页顶部的导入框即可自动分析
