Cursor 用量本地采集器

适合 Cursor Pro / Team 成员，没有 Admin API 权限时使用。它只读取你当前浏览器里 Cursor dashboard 已显示的汇总数字，不读取 cookie、session、密码或 API key。

安装

把下面这个按钮拖到浏览器书签栏：

Import Cursor Usage

如果浏览器不允许拖拽，也可以右键复制链接地址，手动新建书签并粘贴到 URL。

使用

  1. 打开 Cursor 用量页：cursor.com/dashboard。
  2. 如果你想导入某一天 token，先把鼠标悬停到图表对应日期，让 tooltip 显示 Daily total 和 Cumulative total。
  3. 点击书签栏里的 Import Cursor Usage。
  4. 它会打开 Tea X 首页，用 URL fragment 导入数据到 localStorage，然后自动清掉 fragment。

安全边界

  • 不会读取或上传 Cursor cookie、session、密码、API key。
  • 导入数据通过 #ai_usage=... fragment 传递，fragment 不会发送给 GitHub Pages 服务器。
  • 数据最终只存在你浏览器的 localStorage。
  • 公开主页源码里不保存你的真实金额、token 或模型明细。

能采集什么

  • 月度金额额度和本月已花费。
  • 当前 7 天 Total spend，并换算日均花费。
  • 图表 tooltip 可见时的 Daily total tokens 和 Cumulative total tokens。
  • 账期重置日。

如果 Cursor 页面结构变化，书签可能需要更新。
